#### 주요 기능
* 한 명씩 번갈아가며 투사체를 던집니다.
* 게이지가 찬 만큼 강하게 던집니다.
* 각도를 조절하여 던질 수 있습니다.
* 투사체는 랜덤으로 생성되는 바람에 영향을 받습니다.
* 부위별 입는 피해량이 다릅니다.

#### 차후 추가될 기능
* 다양한 아이템 옵션
